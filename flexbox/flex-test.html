What is Flexbox?

‚è±Ô∏è Estimated time: 5 minutes

CSS Flexbox (Flexible Box Layout) is a layout method that provides an efficient way to arrange, distribute, and align elements in a container, even when their size is unknown or dynamic.

Key Terminology:

  • Flex Container: The parent element with display: flex
  • Flex Items: The direct children of the flex container
  • Main Axis: The primary axis along which flex items are laid out
  • Cross Axis: The axis perpendicular to the main axis

Individual Item Control

‚è±Ô∏è Estimated time: 8 minutes

Control individual flex items with these properties:

.item { flex-grow: 1; /* How much should the item grow? */ flex-shrink: 1; /* How much should the item shrink? */ flex-basis: auto; /* What's the initial size? */ /* Shorthand: flex: grow shrink basis */ flex: 1 1 auto; }

Lesson Summary

Key Takeaways:

  • display: flex creates a flex container
  • flex-direction sets the main axis direction
  • justify-content aligns items along the main axis
  • align-items aligns items along the cross axis
  • flex-wrap controls line wrapping
  • flex property controls item growth and shrinking
